# uxsalt

A Claude Code plugin marketplace for LLM-first UX tooling.

## Install the marketplace

```
/plugin marketplace add jakecjones/uxsalt
```

## Available plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [ux-harness](./ux-harness) | 2.3.0 | LLM-first design system and component standard for React/Tailwind frontends. 11 skills and 5 agents for building, auditing, and migrating UI. |

## Install a plugin

```
/plugin install ux-harness@uxsalt
```
